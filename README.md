## Setting up development environment
```
$ git clone https://github.com/inventit/npm-moat.git
$ cd npm-moat
$ git checkout develop
$ npm update -g npm
$ npm install -g grunt-cli
$ npm install -g mocha
$ npm install --save-dev
```
